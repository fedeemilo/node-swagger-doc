# Autogenerating Swagger Documentation with Node & Express

## Tools to learn

- Swagger: will help us to document our api's. It also provides a user interface that will
  help us to test our api's.

  Documentation: https://swagger.io/specification

- swagger-jsdoc: an npm module that help's us document our code and generates swagger documentation
  from the comments in our code. This is what gives the auto generation of our documentation.

  Documentation: https://www.npmjs.com/package/swagger-jsdoc

- swagger-ui-express: another npm module that will take the documentation that's generated by
  swagger-jsdoc and allow it to be served using an express API server. So this gives us live updating
  API documentation in a user interface and that user interface can also be used to test the running
  application

  Documentation: https://www.npmjs.com/package/swagger-ui-express

### Installing dependencies

```
npm install swagger-jsdoc swagger-ui-express
```

### Require definitions and usage

```javascript
const swaggerJsDoc = require("swagger-jsdoc");
const swaggerUi = require("swagger-ui-express");

/* Define swagger options */
const swaggerOptions = {
  swaggerDefinition: {
    info: {
      title: "Customer API",
      description: "Customer API Information",
      contact: {
        name: "WePlan"
      },
      servers: ["http://localhost:5000"]
    }
  },
  apis: ["app.js"]
};

const swaggerDocs = swaggerJsDoc(swaggerOptions);
app.use("/api-docs", swaggerUi.serve, swaggerUi.setup(swaggerDocs));


```

### Example of route comment documentation


```javascript
/**
 * @swagger
 * /customers:
 *  get:
 *    description: Use to request all customers
 *    responses:
 *      '200':
 *        description: A succesful response
 */
app.get("/customers", (req, res) => {
  res.send("Customer results");
});
```


 ```javascript
/**
 * @swagger
 * /customer:
 *  put:
 *    description: Use to update a customer
 *    responses:
 *      '201':
 *        description: A succesful response
 */
 app.put("/customer", (req, res) => {
  res.status(200).send("Succesfully updated customer");
});
 ```
